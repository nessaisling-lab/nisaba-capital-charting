Behavior rules

  • fetching_pill → become generic Pill struct. Variants: Sparkly · Alert · Transit · Error · Success · Info.
  • Stack of pills sits between right-spacer and gear. Max 3 visible at once. Older pills dismissed on TTL or fade out.
  • Each pill carries optional on_press → routes to relevant tab.
  • Sparkly variant uses existing ShootingStar canvas. Other variants use static Phosphor glyph with optional pulse animation.
  • TTL defaults: fetch = lives while fetching, alert = 8s, transit = 12s, error = 15s (or click-to-dismiss), success = 4s.
  • All notifications also append to notification_history drawer (bell or gear-click reveal). Drawer overlays via stack! — never pushes layout.
  • Theme: dark parchment bg, gold border, drop shadow. Already matches existing fetch_pill + alert_pill style.
